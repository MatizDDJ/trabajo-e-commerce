 # Trabajo E-commerce

Esta es una aplicación de ejemplo de e-commerce desarrollada con React. Contiene listado de productos, detalle y carrito de compras.

 Tecnologías principales:
- React (v19)
- React Router
- Bootstrap
- React Toastify
- FontAwesome

 Características:
- Navegación entre páginas con `react-router-dom`.
- Carrito de compras (estado en memoria; comprobar si está persistido en `localStorage` según la implementación).
- Notificaciones con `react-toastify`.

 Cómo ejecutar localmente:

```powershell
npm install
npm start
```

Comandos útiles:

```powershell
npm run build   # crear build de producción
npm test        # ejecutar tests
```

Estructura relevante:
- `src/` — código fuente (componentes, estilos, entradas).
- `public/` — activos estáticos y `index.html`.
- `package.json` — dependencias y scripts.
